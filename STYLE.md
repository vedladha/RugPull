# Style

this style guide is the general style guide for a project.

## Java

the [checkstyle](https://checkstyle.org/) plugin will be used within gradle to enforce this.

Java code should follow the [Google java guidelines](https://google.github.io/styleguide/javaguide.html). As specified by
Google. The full list of Google checks can be found [here](https://github.com/checkstyle/checkstyle/blob/master/src/main/resources/google_checks.xml).
